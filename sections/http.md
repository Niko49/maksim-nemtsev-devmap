# Протокол HTTP

## Цель
Понять, как браузер взаимодействует с веб-сервером. Для этого необходимо познакомиться с такими понятиями как:
- User-agent
- DNS
- Отличия HTTP 1.1 от версии 1.0
- Тело HTTP-запроса
- Передача данных
- Отправка форм
- Transfer-Encoding
- Перенаправления
- Базовая аутентификация
- Cookies
